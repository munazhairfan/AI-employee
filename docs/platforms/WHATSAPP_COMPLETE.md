## Complete Flow

```
1. WhatsApp Message Received (with keywords: urgent, invoice, payment, etc.)
        ↓
2. WhatsApp Watcher (watchers/whatsapp_watcher.py)
   - Monitors WhatsApp Web
   - Detects unread messages with keywords
   - Creates: Needs_Action/WHATSAPP_timestamp.md
        ↓
3. Orchestrator (src/orchestrator.py)
   - Picks up Needs_Action file
   - Analyzes message
   - Creates draft: Pending_Approval/WhatsApp_Draft_timestamp.md
        ↓
4. Human Review
   - Opens draft file
   - Reviews suggested reply
   - Checks "Approve" box
        ↓
5. Orchestrator Polling
   - Detects approval
   - Calls: POST http://localhost:3006/send_whatsapp
   - Includes: to, message, draft_file
        ↓
6. WhatsApp MCP Server (src/whatsapp_mcp.js)
   - Sends via Twilio or Meta API
   - Moves draft to Done/
        ↓
7. Complete!
```

## Approval Workflow

### To Approve:

1. Open the draft file in Obsidian/text editor
2. Change to: `- [x] Approve (execute manually)`
3. Save the file
4. Orchestrator polls every 30 seconds
5. Detects approval → Sends via MCP
6. Moves file to `Done/`

### To Reject:

1. Open the draft file
2. Change to: `- [x] Reject (add your reason below)`
3. Add reason: `Reason: Not urgent, will handle later`
4. Save
5. Orchestrator moves to `Rejected/`
