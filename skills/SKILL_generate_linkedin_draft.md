## Description: Generate LinkedIn post draft for human review (draft-only, no API calls).

## Prompt:
Using filesystem tools only, read Dashboard.md and all .md files from vault_path='AI_Employee_Vault/Needs_Action'. Search for business keywords: 'sales', 'business', 'project', 'lead', 'service', 'opportunity', 'client', 'invoice', 'payment', 'milestone', 'achievement'.

If keywords found, generate professional LinkedIn post:
- Length: 200-500 characters
- Include 2-3 relevant hashtags
- End with call-to-action (e.g., "DM me for details!")
- Tone: Professional, engaging, personalized to content

Create draft file in /Pending_Approval: LinkedIn_Draft_{datetime.now().strftime('%Y-%m-%d_%H-%M-%S')}.md

File format:
```markdown
---
type: linkedin_draft
status: pending_review
generated_at: {ISO timestamp}
source_file: {triggering file name}
---

## Post Text

{Generated LinkedIn post content}

---

## Approval Actions

- [ ] Approve (post manually to LinkedIn)
- [ ] Reject (add your reason below)
- [ ] Edit (add your changes/notes below)

---
*Draft generated by SKILL_generate_linkedin_draft.md - No API calls made*
```

If no business activity found, output: "No LinkedIn draft needed at this time."

**CONSTRAINTS:**
- Do NOT call any MCP endpoints
- Do NOT post to LinkedIn
- Do NOT use external APIs
- Output ONLY the draft file content
- No additions, no hallucinations
