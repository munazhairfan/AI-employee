## Description: Generate Facebook post draft for human review (draft-only, no API calls).

## Prompt:
Using filesystem tools only, read Dashboard.md and all .md files from vault_path='AI_Employee_Vault/Needs_Action'. Search for business keywords: 'sales', 'business', 'project', 'lead', 'service', 'opportunity', 'client', 'milestone', 'achievement', 'launch', 'announcement'.

If keywords found, generate professional Facebook post:
- Length: 150-300 characters (Facebook optimal)
- Include 2-3 relevant hashtags
- Include emoji if appropriate (🎉, 📈, 💼, etc.)
- End with call-to-action (e.g., "Learn more!", "Message us!", "Visit our page!")
- Tone: Friendly, engaging, shareable

Create draft file in /Pending_Approval: Facebook_Draft_{datetime.now().strftime('%Y-%m-%d_%H-%M-%S')}.md

Draft format:
```markdown
---
type: facebook_draft
status: pending_review
generated_at: {ISO timestamp}
source_file: {triggering file name}
---

## Post Text

{Generated Facebook post content with emojis and hashtags}

---

## Approval Actions

- [ ] Approve (post manually to Facebook)
- [ ] Reject (add your reason below)
- [ ] Edit (add your changes/notes below)

---
*Draft generated by SKILL_post_fb_draft.md - No API calls made*
```
