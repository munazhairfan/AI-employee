## Description: Generate WhatsApp reply draft for human review (draft-only, no API calls).

## Prompt:
Using filesystem tools only, read all .md files from vault_path='AI_Employee_Vault/Needs_Action' with type 'whatsapp' in frontmatter. Generate WhatsApp reply drafts for human review and approval.

### Step 1: Identify WhatsApp Messages

Scan all .md files in /Needs_Action for:
- `type: whatsapp` in frontmatter
- Files matching pattern: `WHATSAPP_*.md`
- Content from WhatsApp Web scraper

### Step 2: Analyze Message Content

For each WhatsApp message, extract:
- **From:** Sender name/number
- **Message:** Original message text
- **Timestamp:** When message was received
- **Keywords:** 'urgent', 'asap', 'invoice', 'payment', 'help', 'question', 'meeting', 'call'

### Step 3: Generate Reply Draft

Create draft file in /Pending_Approval: WhatsApp_Reply_Draft_{datetime.now().strftime('%Y-%m-%d_%H-%M-%S')}.md

Draft format:
```markdown
---
type: whatsapp_reply_draft
status: pending_review
generated_at: {ISO timestamp}
source_file: {original WhatsApp file name}
---

## Original Message

**From:** {sender name/number}
**Received:** {timestamp}
**Message:**
{Original message text}

## Suggested Reply

{Generated WhatsApp reply - casual, concise, friendly tone}

---

## Approval Actions

- [ ] Approve (send reply manually via WhatsApp)
- [ ] Reject (add your reason below)
- [ ] Edit (add your changes/notes below)

---
*Draft generated by SKILL_generate_whatsapp_draft.md - No API calls made*
```

### Reply Guidelines

**For Invoice/Payment Queries:**
```
Hi! Thanks for reaching out. Regarding your invoice query, let me check the details and get back to you within the hour. Appreciate your patience! 🙏
```

**For Urgent Requests:**
```
Got your message! I'm on it. Let me look into this right away and will update you shortly. Thanks for flagging! 👍
```

**For Meeting Requests:**
```
Hi! Thanks for the meeting request. Let me check my calendar and confirm a time that works. Will revert by EOD. Looking forward to it! 📅
```

**For General Inquiries:**
```
Hi! Thanks for your message. I'd be happy to help with that. Let me get the details and circle back to you soon. Anything specific you'd like me to prioritize? 💬
```

### Step 4: Log Draft Creation

Append to /Dashboard.md:
```markdown
## WhatsApp Reply Drafts - {timestamp}

| From | Status |
|------|--------|
| {sender} | Pending Approval |
```

### Edge Cases

| Scenario | Handling |
|----------|----------|
| **Multiple messages from same person** | Combine into single reply draft |
| **Message unclear** | Draft clarification request |
| **Spam/promotional** | Draft polite decline, flag for review |
| **Already replied** | Skip draft creation |

**CONSTRAINTS:**
- Do NOT call any MCP endpoints
- Do NOT send WhatsApp messages automatically
- Do NOT access WhatsApp API
- Keep replies casual and concise (WhatsApp style)
- Output ONLY the draft file content
- No additions, no hallucinations
