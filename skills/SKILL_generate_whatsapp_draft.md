## Description: Generate WhatsApp reply draft for human review with anti-hallucination safeguards (draft-only, no API calls).

## ANTI-HALLUCINATION RULES (CRITICAL)

1. **Extract ONLY** - Never invent names, numbers, amounts, dates, or facts
2. **Cite Sources** - Every extracted field must include exact source text
3. **Confidence Score** - Rate confidence (0-100%) for each extraction
4. **Mark Unknown** - If info not found, mark as "UNKNOWN - needs clarification"
5. **No Assumptions** - Do not assume sender identity, intent, or context

## Prompt:
Using filesystem tools only, read all .md files from vault_path='AI_Employee_Vault/Needs_Action' with type 'whatsapp' in frontmatter. Generate WhatsApp reply drafts for human review and approval.

### Step 1: Identify WhatsApp Messages

Scan all .md files in /Needs_Action for:
- `type: whatsapp` in frontmatter
- Files matching pattern: `WHATSAPP_*.md`
- Content from WhatsApp Web scraper
- **Record exact source file path for each message**

### Step 2: Analyze Message Content with Source Citations

For each WhatsApp message, extract with citations:

| Field | Extraction Rule | Source Citation Required |
|-------|-----------------|-------------------------|
| **From:** | Sender name/number | Quote exact text showing sender |
| **Message:** | Original message text | Copy verbatim from source |
| **Timestamp:** | When received | Quote timestamp from source |
| **Keywords:** | 'urgent', 'asap', 'invoice', 'payment', etc. | Quote exact words |
| **Intent:** | What sender wants | Quote text implying intent |
| **Sentiment:** | Positive/Neutral/Negative | Quote words indicating tone |

**Confidence Scoring:**
- 100%: Explicitly stated (e.g., "This is John from ABC Corp")
- 90-99%: Strongly implied (e.g., known number + context)
- 70-89%: Reasonable inference (e.g., "urgent" → needs quick response)
- 50-69%: Weak inference (e.g., vague request)
- 0-49%: Unknown/not mentioned

### Edge Cases with Anti-Hallucination Guards

| Scenario | Handling |
|----------|----------|
| **Multiple messages from same person** | Combine into single draft, cite all sources |
| **Message unclear** | Draft clarification request, flag for human review |
| **Spam/promotional** | Draft polite decline, flag for review (confidence <50%) |
| **Already replied** | Skip draft creation (check conversation history) |
| **Unknown sender** | Mark sender as "UNKNOWN", ask human to identify |
